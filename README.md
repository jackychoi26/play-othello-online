# play-othello-online &middot; [![License - MIT](https://img.shields.io/badge/License-MIT-blue)](#license)

`play-othello-online` is an othello online platform in `Next.js`.

## Installation

1. Install via `npm install`
2. Run `npm run dev`
3. Access `localhost:3000`
